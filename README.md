## About

A cinematic, chapter-based portfolio built as a **3D rollercoaster ride through statistical distribution curves** — Normal, F, t, Regression, and Neural Pathway. Each chapter of the story lives on a different curve, and you ride between them.

Built entirely as a single HTML file with no build tools or frameworks.

---

| Layer | Tools |

## Hosting Steps

### 1. Create the repository

```bash
# Go to github.com → New repository
# Name it exactly: portfolio
# Set to Public
# Do NOT initialise with README (you already have one)
```

### 2. Clone and add files

```bash
git clone https://github.com/Obedientononline/portfolio.git
cd portfolio

# Copy your portfolio file in and rename it
cp path/to/Shivani_portfolio_v6.html index.html

# Add the README
cp path/to/README.md README.md
```

### 3. Push to GitHub

```bash
git add index.html README.md
git commit -m "🎢 Launch: Statistics Station portfolio v6"
git push origin main
```

### 4. Enable GitHub Pages

```
GitHub repo → Settings → Pages
Source: Deploy from a branch
Branch: main / (root)
Click Save
```

### 5. Your live URL

```
https://obedientononline.github.io/portfolio
```

> Takes 1–2 minutes to go live after first push. Update the LinkedIn post link once it's up.
