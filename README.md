## About

A cinematic, chapter-based portfolio built as a **3D rollercoaster ride through statistical distribution curves** Normal, F, t, Regression, and Neural Pathway. Each chapter of the story lives on a different curve, and you ride between them.

Built entirely as a single HTML file with no build tools or frameworks.

---

## Tech Stack

| Layer | Tools |
|---|---|
| 3D Engine | Three.js |
| Animation | GSAP |
| Skills Graph | Canvas API (custom constellation) |
| Projects | GitHub REST API |
| Fonts | Cormorant Garamond, DM Sans, Caveat, JetBrains Mono |
| Hosting | GitHub Pages |

---

## Chapters

1. **Getting to Know Me** - The story so far, on a Normal distribution
2. **Explore Work** - Live GitHub projects on an F-distribution
3. **Skills & Tools** - Interactive constellation graph on a t-distribution
4. **Building with Curiosity** - Experience timeline on a Regression curve
5. **Enter Playground** - Curiosity cards on a Neural Pathway
